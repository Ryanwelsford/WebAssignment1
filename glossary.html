Glossary - Media Special

Meta tag

The meta tag enables you to declare various properties of your web page. Including the character set you are using, typically UTF-8, the default language of your web page and the ability to refresh your page after a certain number of seconds or enables redirection to a separate URL after a set time. Must be declared within the head tag!.

Viewport

“The browser's viewport is the area of the window in which web content can be seen. This is often not the same size as the rendered page, in which case the browser provides scrollbars for the user to scroll around and access all the content.”

Courtesy of Mozilla

Initial-scale

The initial-scale property controls the zoom level when the page is first loaded. Essentially controls how far away the page appears on first load, it can be set to different values depending on what type of device is loading the web page, i.e. mobile or desktop.

List-style-type

List-style-type enables you to remove bullet points from lists by setting its value to none. It can also be set to a variety of different options to give your lists a different look than standard, try setting to lower-roman for roman numerals, as you may have seen today already.

Media

Used through the @media tag rather than standard tags, enables conditional rules of web pages based on the pixel size of the view window. In combination with setting max-width and min-width enables media queries on your web page! You can try setting different values for elements like your header, reducing the font size maybe, and watch as you shrink the page your header will shrink at the pixels sizes you set.

Text-decoration

Text-decoration allows you to decorate your text in different ways, obvious right! The best use so far I have found is, it enables you to remove the blue highlight and underline from hyperlinks for things like your social media links in your footers! You can also do fun things like set a green wavy line under words.Hover over the heading!

Favicon

A favicon is the small icon/image on web pages tabs appearing next to the "title" element, typically used for small logos. Look at your tab for the Rw icon

Transform

Transform lets you change elements of your code through various means. The scale value allows you to change the size of an element by a certain factor, larger than one for increases in size and decimal values for decreasing size. You can even rotate objects in 3D using transform using the rotate3d value and inputting three values for dimensions and a final value for an angle, remember to separate your dimensions with commas! Hover over the heading!

Transition

The transition property works with the transform property to allow you to adjust how your elements change. For instance, you can set a transform property to have a transition of 0.5 meaning that it will take 0.5 seconds for the change to fully complete. You can also set it to things like linear, which makes it change equal amounts over time. Try setting to step(3, end) your transition will jump through its transform over 3 distinct steps.
