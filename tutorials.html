Media Queries

This video has taken us through the basics for us to start using media queries, the best way I have found to think about them is to think of them like IF statements. Essentially what the query says is what to do when the pixel size of the viewport is x size. Sounds like an IF statement to me at least.

For instance, setting max-width: 480px; is saying when the screen is less than 480px do this. As an example, on this site if you reduce the screen size to 480px quite a lot happens, it moves into a mobile friendly format that changes the flex direction from row, allowing us to stack articles next to each other, to column stacking them on top of each other.
